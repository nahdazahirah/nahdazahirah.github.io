Beberapa hasil karya saya

Project 1: Programming

Ini adalah salah satu contoh proyek coding yang pernah saya kerjakan menggunakan bahasa C#. Proyek ini saya buat untuk melatih logika dasar dan pemahaman alur program, sekaligus memperluas kemampuan saya di bidang pemrograman.

Project 2: Robot Avoider

Robot Avoider adalah robot sederhana yang saya buat menggunakan Arduino dan sensor ultrasonik. Cara kerjanya cukup simpel: saat sensor mendeteksi ada benda di depannya, robot ini akan otomatis berhenti. Proyek ini jadi salah satu pengalaman seru saya dalam eksplorasi robotik.

Project 3: Hand Sanitizer Otomatis

Alat ini saya rancang untuk bisa mengeluarkan hand sanitizer secara otomatis begitu tangan terdeteksi oleh sensor. Dibuat dengan Arduino, proyek ini jadi bentuk kecil dari solusi praktis yang bisa diterapkan dalam kehidupan sehari-hari.

Project 4: UI/UX Mockup

Di sini saya menampilkan beberapa hasil latihan desain antarmuka, baik untuk versi desktop maupun mobile. Fokusnya adalah bikin tampilan yang enak dilihat dan gampang digunakan. Desain ini saya buat untuk tugas mata kuliah Interaksi Manusia dan Komputer.

Project 5: Photograph

Fotografi jadi cara saya menangkap momen dan hal-hal menarik yang sering luput dari perhatian. Gambar-gambar di sini saya ambil sebagai bentuk ekspresi dan dokumentasi dari sudut pandang saya sendiri.

Project 6: Drawing

Saya suka menggambar, terutama sketsa rumah dan bangunan. Lewat coretan-coretan ini, saya bisa menuangkan imajinasi dan ketertarikan saya pada bentuk, struktur, dan detail arsitektur.

Project 7: Baking

Kalau lagi jenuh atau butuh istirahat dari urusan kampus, saya biasanya baking. Hobi ini jadi pelarian santai yang bikin hati tenang, sekaligus bikin dapur jadi lebih wangi.
